# api-1803

API REST desenvolvida com:

- Node.js
- Express
- TypeScript
- MongoDB

Arquitetura: Monólito Modulas com camadas.

Etapas para criação da API:

1 - Criar repositório e configurar projeto PC
2 - Buscar o comando na doc para iniciar um projeto node.
3 - Instalar bibliotecas:
        - Express
        - Type de Express
        - Types do Node
        - typescript
        - tsx
        - dotenv
        - cors
        - types cors
        - Mongoose

4 - Criar o arquivo tsconfig
5 - Ajustar o arquivo tsconfig de acordo com o projeto.
6 - Montar a arquitetura: Monólito modular com camadas.
7 - Criar script para rodar o Projeto (Package.json)
8 - Codar o arquivo app.ts
9 - Codar Route.ts
10 - Codas server.ts
11 - Codar .env
12 - Criar rota de teste e fazer uma requisição no postman
